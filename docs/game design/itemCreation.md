Current iteration of Item Creation is a simplified version of Star Ocean 2's. However, it doesn't necessarily fit the narrative theme of the game.

To help resolve this I came up with the idea that monsters each have only one type of item creation accessible to them - and, most of the time, it doesn't match their strongest stat either.
So you'll want a diverse party in order to have ample access to Item Creation, and while some creatures might excel at Item Creation they might be poor in battle and vice-versa.

Menu flow will me Item Creation -> Select Creature -> Item Creation menu.
