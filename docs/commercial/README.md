# Commercial Release Pipeline

This folder turns the game's commercial identity into work that can be planned,
tested, and shipped. It is intentionally separate from engine specifications and
authored campaign notes.

- [`release-plan.md`](release-plan.md): stage gates, evidence, and delivery
  sequence from proof build to post-launch.
- [`proof-build.md`](proof-build.md): the smallest playable build that can prove
  the game's emotional and commercial loop.
- [`store-positioning.md`](store-positioning.md): working public-facing copy,
  capsule/trailer rules, tags, and content pillars.

The governing product direction lives in
[`../design/commercial-identity.md`](../design/commercial-identity.md).
